Interact is a module that contains all the necessities for a terminal-based program.

<var>(*list) = [] : Accepts list.

[<arg>=<var>] : Optional argument.

Functions : 

1.msg(m=*string , [f=0],[e='']) : Display a message to the user. f=1 - Flush=True , e='.' - No newline , end line with '.'
2.star(m=*string , [f=0],[e='']) : Display a starred message to the user. f=1 - Flush=True , e='.' - No newline , end line with '.'
3.error(m=*string) : Display an error message to the user.
4.warn(m=*string) : Display a warning to the user.
5.info(m=*string) : Display an info box to the user.
6.tip(m=*string) : Display a tip to the user.
7.box([title=*string],m=*string) : Display a boxed message to the user. title : Displays title of the box.

Best part about sql connectivity using this module, you don't need to import any sql module. It is auto-connected.

8.sign_up([sqlhost=*string] , sqluser=*string , sqlpass=*string , outdb=*string , [dbtable=*string]) : Registers a user by prompting credentials,assigns a unique m_id to the user and stores the credentials and m_id
                                                                                                       in the desired table. Default table name : credentials , default host : localhost

9.login([sqlhost=*string] , sqluser=*string , sqlpass=*string , checkdb=*string , [dbtable=*string]) : Helps a user to login by prompting credentials and validates them by checking in the desired table.
                                                                                                       Default table name : credentials , default host : localhost

10.get_items([sqlhost=*string] , sqluser=*string , sqlpass=*string , checkdb=*string , dbtable=*string, [reqcol=*string]  , [wherecol=*string], [pattern=*string]) : Selects and returns items from desired column of desired table using desired conditions. 
                                                                                                                                                                                  Default reqcol = * , default host : localhost
                                         
11.update([sqlhost=*string] , sqluser=*string , sqlpass=*string , checkdb=*string , dbtable=*string ,reqcol=*string, newdata=*anydatatype [wherecol=*string] , [pattern=*string]) : Updates item of desired column from desired table using desired 
         																                       conditions. Default host : localhost
12.multi_update([sqlhost=*string] , sqluser=*string , sqlpass=*string , checkdb=*string , updates(*list)=[
                                                                        (<table1>,<reqcol1>,<newdata1>,[<wherecol1>],[<pattern1>]),
                                                                        (<table2>,<reqcol2>,<newdata2>,[<wherecol2>],[<pattern2>]), ...
                                                               ... (<table n>,<reqcol n>,<newdata n>,[<wherecol n>],[<pattern n>])
                                            ])  :   Updates all items of desired column from desired table using desired conditions.
                                                    Default host : localhost
                                                    
13.clr() : Clear terminal.

14.menu(title=*string , items(*list)=[
                                           <item1>
                                           <item2> ...
                                           ... <item n>
                                     ], [reuse=0])        : Creates a menu with the desired title and items. Default reuse : 0. If reuse=1, menu function will return a string. It can be stored in a variable.


15.prompt_(m=*string) : Prompts the user. Returns a string.

16.yesno() : Asks the user for a yes/no confirmation. Returns an integer ('y' - 1 , 'n' - 0 , default - 2).

17.exit_yesno() : Asks the user for a yes/no confirmation when he/she wants to exit. Returns an integer ('y' - 1 , 'n' - 0 , default-2).

18.custom_yesno(m=*string) : Asks the user for a yes/no confirmation with a custom prompt. Returns an integer ('y'-1 ,'n'-0, default-2).

Even file maipulation is automated. No need to import any os module.

19.readfile(filepath=*string , [return_="whole"],[amt=*int]) : Returns file contents. return_="lines" - Returns a list of lines in the file. amt - ("whole" : gives first *amt characters of the file. "lines" : gives first *amt lines of the file in a list.)

19.writefile(filepath=*string, newdata=*anydatatype , [mode="append"]) : Writes data into a file. mode="overwrite" : Overwrites file
                                                                                                                     contents.
                                                                                                                     
20.deletefile(filepath=*string) : Deletes the file.

21.createfile(filepath=*string) : Creates the file.

22.runfile(filepath=*string) : Runs the file.

23.pause([min_=*int] , [sec=*int] , [msec=*int]) : Pauses the program for specified time.

21.copyfile(filepath=*string,copypath=*string) : Copies the file to the copypath path.

21.movefile(filepath=*string,newpath=*string) : Moves the file to the newpath path.
